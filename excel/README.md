🛒 Excel Project - Sales Management

This project consists of an Excel spreadsheet developed to manage and analyze store sales, using advanced features such as data validation, automated formulas, and pivot tables.

📌 Spreadsheet Structure

The file is composed of 4 main sheets:

📋 Menu
Home screen for navigation
Provides easy access to the other sheets in the workbook
💰 Sales
Area for recording completed sales
The user selects the product through a dropdown list (data validation)
Automatic filling of the following information:
🆔 Product ID
💲 Price
🏷️ Category
These fields are automatically populated using the VLOOKUP function
⏱️ The date and time are automatically recorded at the exact moment the product is entered and remain fixed after insertion
📦 Product List
Database containing:
Product ID
Product name
Category
Price
Serves as the reference for data validation and VLOOKUP
📊 Analysis
Provides an overview of sales performance
Uses:
📈 Pivot Table
📊 Pivot Chart
Enables analysis of:
Total sales
Best-selling products
Top-performing categories
⚙️ Features Used
✅ Data Validation (product dropdown list)
✅ VLOOKUP function for automatic data retrieval
✅ Automatic fixed date and time registration
✅ Pivot Table for data analysis
✅ Pivot Chart for data visualization
🎯 Project Objective

To demonstrate practical Excel skills, including:

Data organization
Process automation
Creation of simple dashboards
Sales data analysis

🛒 Projeto Excel - Controle de Vendas

Este projeto consiste em uma planilha de Excel desenvolvida para gerenciar e analisar as vendas de uma loja, utilizando recursos avançados como validação de dados, fórmulas automáticas e tabelas dinâmicas.

📌 Estrutura da Planilha

O arquivo é composto por 4 abas principais:

📋 Menu
Tela inicial de navegação
Facilita o acesso às demais abas da planilha
💰 Vendas
Área de registro das vendas realizadas
O usuário seleciona o produto através de uma lista suspensa (validação de dados)
Preenchimento automático das seguintes informações:
🆔 ID do produto
💲 Preço
🏷️ Categoria
Esses dados são retornados automaticamente utilizando a função PROCV
⏱️ A data e hora são registradas automaticamente no momento em que o produto é inserido, permanecendo fixas após o preenchimento
📦 Lista de Produtos
Base de dados contendo:
ID do produto
Nome do produto
Categoria
Preço
Serve como referência para a validação de dados e para o PROCV
📊 Análise
Apresenta uma visão geral das vendas
Utiliza:
📈 Tabela Dinâmica
📊 Gráfico Dinâmico
Permite análise de:
Total de vendas
Produtos mais vendidos
Categorias com maior desempenho
⚙️ Funcionalidades Utilizadas
✅ Validação de Dados (lista suspensa de produtos)
✅ Função PROCV para preenchimento automático
✅ Registro automático de data e hora fixa
✅ Tabela Dinâmica para análise de dados
✅ Gráfico Dinâmico para visualização
🎯 Objetivo do Projeto

Demonstrar habilidades práticas em Excel, incluindo:

Organização de dados
Automação de processos
Criação de dashboards simples
Análise de informações de vendas
